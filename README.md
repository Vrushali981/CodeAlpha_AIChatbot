# AI Chatbot

A Java chatbot that uses rule-based NLP to respond to user messages.

## Features
- Understands greetings, jokes, Java questions, and more
- Solves basic math (e.g. what is 5 * 4)
- Interactive console conversation

## How to Run

javac AIChatbot.java
java AIChatbot


## Tech Used
Java, Regex, OOP, Pattern Matching
